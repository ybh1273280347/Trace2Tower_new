# Curated Experiment Documentation

Only final experiment conclusions and frozen, auditable comparison contracts
are kept here. Historical diagnostics and invalid runs remain in their original
experiment directories.

- `REPORT.md` is the Chinese main experiment report.
- `CROSS_BENCHMARK_RESULTS.md` is the compact cross-domain result summary.
- `CROSS_MODEL_PAIRED_ANALYSIS.md` is the paired cross-model and author-executor analysis.
- `REVIEW_SUPPLEMENT.md` collects the requested clarification and existing-data analyses.
- `../experiments/webshop/expert-crafted-v1/RESULTS.json` records the frozen
  WebShop Expert-Crafted Skills validation run and paired comparisons.
- `../experiments/alfworld/official/ALFWORLD_ABLATION_RESULTS.json` records the
  corrected ALFWorld ablation results and invalidated historical D1 run.
- `../experiments/alfworld/official/SKILLX_REWRITE_AUDIT.md` records the exact
  open-source rewrite-and-inject behavior and the ALFWorld runtime diagnostic.
