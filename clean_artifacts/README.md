# Curated Reusable Artifacts

This directory is a copy-only curated set.  Nothing under `artifacts/` was
moved or deleted.  The inventory maps every retained object to its original
source and states exclusions that must not be reused.

The directory contains only input pools, frozen manifests, final skill
libraries, final Tower snapshots, and valid build-ablation snapshots.  It
does not duplicate execution logs or diagnostic result runs.
