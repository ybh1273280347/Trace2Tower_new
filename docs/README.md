# Trace2Tower 文档

当前 WebShop 实验只以 `experiments/webshop/event-tower-v2/` 下的协议、清单和冻结记录为准。

- `PROTOCOL.md`：方法、数据、模型、指标、统计和 artifact 契约。
- `EXPERIMENTS.md`：按阶段执行的实验清单、门控规则和停止条件。
- `configs/experiments/webshop_event_tower_v2.json`：样本划分及机器可读实验计划。

旧方法名、旧验证协议、旧结果说明和实现文档统一归档在 `docs/deprecated/legacy-v1/`，只用于历史追溯，不构成当前实验契约。
